Tools of Choice

Here are a list of skills/frameworks that I use on a regular basis and how I use them to solve problems.

HTML CSS Javascript

Can't be a UI expert without a good foundation in these. I have been using these technologies for over 16 years.

React / React Native

React is a my framework of choice for coding Javascript web applications. React's component architecture makes it easy to reuse code when developing user interfaces. JSX then allows you to combine your view and styles with each component in a simple way. React Native helps development teams create native mobile applications quickly by leveraging their Javascript skills with a familiar framework, all of this while leveraging the power of newer Javascript ES6 features, such as classes, inheritance and module loading.

Bootstrap 3

I prefer Bootstrap 3 for making smaller web sites responsive. Bootstrap 3 is ideal in situations were the customer does not require a high level of customization but can live with the standard components that come with bootstrap.

Bourbon Neat

Increasingly I turn to Bourbon Neat when customers require their site to be both responsive and highly customized. Bourbon Neat makes your HTML less cluttered with classes, since you control your grid layout through SASS mixins.

Node.js

With its familiar syntax, Node.js is fast and equiped with a great package manager (npm). I love to use Node.js to program server side restful services to provide a secure API to the database level.
